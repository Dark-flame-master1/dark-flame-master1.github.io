C语言

一个简单的C程序所包含的内容

1.头文件

通俗的理解来说，类似于将stdio.h的文件中所有的内容复制粘贴到当前文件中

#include <stdio.h> //包含另一个文件

2.注释

//不同的注释风格
/* 不同的注释风格 */

3.main()函数

c程序的基本模块，c程序必定会从main()函数开始执行

int main(void)  //一个简单的C程序
//int是返回“整数”的数据类型，返回给操作系统
//void表示不需要传入函数任何信息

4.花括号

记录了函数体的开始和结束的位置

//函数开始
{

5.声明

int num; //声明变量num
//有效的标识符只能包括小写字母，大写字母，数字和下划线，数字不能作为开头
int num,feet//多条声明

6.赋值

num = 1;
//需要先做声明才能够赋值，等于号为赋值号并不是对比左右是否相等，这里的等于号代表了将右边的数据赋值到左边的变量中。

7.输出函数

c语言中会有很多的输出函数，printf是最常见的一个，此函数在括号内接收到数据，从而将其输出到屏幕中

printf("I am a simple");
printf("computer.\n");
printf("My favorite number is %d because it is first.\n",num);
printf("press Enter to quit the game...");
getchar();

8.return函数

在程序的最后返回出0

    return 0;
}//函数结束

9.函数声明

#include<stdio.h>
void secondFunc(void)
int main(void)
{
    printf("引入另一个函数，他在这里\n");
    secondFunc();
}
void secondFunc(void)
{
    printf("我在这里，我叫secondFunc");
}

总结：一个C程序的结构

#include <stdio.h> 
int main(void)//函数头
//函数体
{
    //声明
    //语句
    return 0;
}
//大部分语句以分号结尾
